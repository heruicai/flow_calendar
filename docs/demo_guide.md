# FlowCal Demo Guide

This document will track demo steps as features are implemented.

## Planned Demo Flow

1. Start the Streamlit app.
2. Add a fixed event with a natural-language command.
3. Add a deadline task and show the deadline timeline.
4. Add an essential task and show the daily bar.
5. Add a flexible plan and show the todo pool.
6. Mark a task completed and confirm that it turns gray.
7. Demonstrate voice simulation using typed text.
