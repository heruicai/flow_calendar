# Sample Commands

Use these commands in the FlowCal demo. All examples are fictional.

## Fixed Event

- 明天下午三点到四点参加算法面试
- 今天上午十点开项目会议
- 下周一下午两点到三点参加组会
- 周五上午九点笔试

## Deadline Task

- 周五前完成报告，预计三小时
- 明晚前提交作业，大概两小时
- 下周二前修改简历，预计30分钟
- 今晚前完成项目提交，需要1.5小时

## Essential Task

- 今天必须洗衣服
- 晚上提醒我买药
- 明天记得取快递
- 今天别忘了交水电费

## Flexible Plan

- 添加弹性任务，刷两道 LeetCode
- 添加弹性任务，复习数据库
- 这周安排一些时间阅读论文
- 添加弹性任务，整理项目资料

## Query Schedule

- 我今天有什么安排
- 我明天有什么安排
- 查看周五日程
- 查看下周一日程

## Update Or Delete

- 洗衣服完成了
- 把报告标记为完成
- 删除明天下午的算法面试
- 把洗衣服删掉

## Clarification Example

- 明天下午提醒我复习

This command may ask for a more specific time because it has a date window but no concrete time.
